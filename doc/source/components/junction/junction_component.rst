Physical Model
==============

The equations added by a junction correspond to a mass flow balance for hydraulic calculations.

.. image:: junction_hyd.png
	:width: 15em
	:alt: alternate Text
	:align: center

.. math::
   :nowrap:

   \begin{align*}
    \sum_{n=0}^N \dot{m}_{in_n} - \sum_{l=0}^M \dot{m}_{out_l} = 0
   \end{align*}

For temperature calculations, the thermal power carried by the incoming fluids is balanced:

.. image:: junction_heat.png
	:width: 15em
	:alt: alternate Text
	:align: center

.. math::
   :nowrap:

   \begin{align*}
    \sum_{n=0}^N \dot{m}_{in_n}c_p T_{in_n} - \sum_{l=0}^M \dot{m}_{out_l} c_p T_{m} = 0
   \end{align*}
